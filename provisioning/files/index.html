Welcome to my LAMP stack!

This project is an implementation of the LAMP-stack (Linux, Apache, MySQL, PHP), hosted on a VM and configured using Vagrant.

Please note that in lieu of MySQL, we use MariaDB (which functions mostly exactly the same for this project). In addition, the PHP-code used in this project does not follow best practices nor is it particularly secure. This is merely experimentation and definitely not intended for use in a production environment.
